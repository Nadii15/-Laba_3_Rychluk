- ### Alert-блок
> [!TIP]
**Совет по использованию:** Для экранирования специальных символов в Markdown используйте обратный слэш \ . Например, чтобы написать *курсив* без форматирования, напишите \ *курсив\ *
- ##  ESCAPE-символы
- Заголовок без форматирования: **\ * Это не заголовок**
- Звёздочка как текст: **\ * Просто звёздочка**\ *
