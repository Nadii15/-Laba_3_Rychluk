Markdown позволяет вставлять фрагменты кода с подсветкой синтаксиса, а также
цитаты.
Блоки кода
Встроенный (inline) код:
Команда `git status` показывает состояние репозитория.

Примеры команды Git в виде inline-кода
``` git add .```

Многострочный код:
```python
print("Hello, Markdown!")
```
Можно указать язык: python, js, bash, csharp и др.
Цитаты
Цитата создаётся символом >:
> Это цитата в Markdown.
Вложенная цитата:
> Первый уровень
>> Второй уровень
